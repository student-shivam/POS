# Screenshot Instructions

Place the project screenshots in this folder with the following names:

## File Names Expected:

1. **01-dashboard.png** - Admin Dashboard (Business Intelligence)
   - Shows revenue cards, order counts, charts
   - Size: ~1400 x 700px

2. **02-pos-menu.png** - Order Menu (POS Interface)
   - Shows item grid with categories
   - Size: ~1400 x 700px

3. **03-inventory.png** - Item List (Inventory Management)
   - Shows inventory table with items
   - Size: ~1400 x 600px

4. **04-settings.png** - Admin Settings (Profile & Configuration)
   - Shows settings page with profile form
   - Size: ~1400 x 700px

## How to Add Screenshots:

1. Take screenshots of each page in your running application
2. Crop/resize to the recommended dimensions
3. Save as PNG format with the exact names above
4. Place in this `docs/screenshots/` folder
5. Commit to GitHub

## When Pushed to GitHub:

The README will automatically display these images when viewed on GitHub, as the image URLs point to:
`https://raw.githubusercontent.com/yourusername/RaVINDRA_POS/main/docs/screenshots/01-dashboard.png`

## Update GitHub URLs:

Replace `yourusername` with your actual GitHub username in the README.md file where it says:
`https://raw.githubusercontent.com/yourusername/RaVINDRA_POS/main/docs/screenshots/`
